Common Conversion problems

  • Avoid other elements like lists and text (should work however)
  • Ensure that your markdown table only has 2 columns
  • Ensure markdown file isn't corrupted
  • Remove the headings as the markdown converter will treat these as flashcards

Flashcard Controls (The controls for the flashcard use the arrow keys)
  • ➡️ for going to the next flashcard (or correct)
  • ⬅️ for going back a flashcard (or Wrong)
  • ⬆️ or ⬇️ to flip the flashcard and show the otherside
  • You can of course use the buttons but it saves time using arrow keys.
