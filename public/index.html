Flint: Sparking your learning!

An electron/web based FOSS flashcard application to solve the problems of existing flashcard applications. Flint solves the problems like clunky UI, poor offline functionality, lack of data exporting control, paywalls and the printing of flashcards. It will be expanded on to be the best flashcard app out there.



This is an inprogress development. Please see the github repositry for further details.

Flint is a free OS (Open Source) project that aims to fix the flaws with existing flashcard applicaitons (like quizlet). The project is an ambitious one. I plan to add:

  • Awesome cloud saving that keeps your flashcards accesible for all users
  • An awesome electron app (same as web app but physical application that works offline)
  • Entertaining games that allow you to learn while having fun and really test your knowledge of the flashcards with a time crunch
  • Printing software that works for double sided and single sided and is easy to make IRL flashcards with.
